1. **What is the difference between null and undefined?**
   `undefined` মানে ভ্যারিয়েবল ডিক্লেয়ার করা হয়েছে কিন্তু কোনো ভ্যালু দেওয়া হয়নি। `null` ইচ্ছাকৃতভাবে খালি বা কোনো ভ্যালু নেই বোঝাতে ব্যবহার করা হয়।

2. **What is the use of the map() function in JavaScript? How is it different from forEach()?**
   `map()` অ্যারের প্রতিটি আইটেমে কাজ করে নতুন একটা অ্যারে রিটার্ন করে। `forEach()` শুধু লুপ চালায়, কিন্তু কোনো নতুন অ্যারে রিটার্ন করে না।

3. **What is the difference between == and ===?**
   `==` ভ্যালু তুলনা করে কিন্তু টাইপ কনভার্ট করে নিতে পারে। `===` ভ্যালু ও টাইপ দুটোই চেক করে, তাই এটা বেশি স্ট্রিক্ট।

4. **What is the significance of async/await in fetching API data?**
   `async/await` ব্যবহার করলে অ্যাসিনক্রোনাস কোড সহজে সিঙ্ক্রোনাসের মতো লেখা যায়। এতে কোড পড়তে ও বুঝতে সুবিধা হয়।

5. **Explain the concept of Scope in JavaScript (Global, Function, Block).**
   Global scope মানে পুরো প্রোগ্রামে অ্যাক্সেস করা যায়। Function scope শুধু নির্দিষ্ট ফাংশনের ভিতরে কাজ করে, আর Block scope `{}` এর ভিতরে সীমাবদ্ধ থাকে।
